# 🌱 SustentaAgro

## 📖 Sobre o Projeto

O **SustentaAgro** é um site educativo e interativo voltado para a conscientização sobre práticas sustentáveis no meio rural. O projeto combina conteúdo informativo com um quiz dinâmico para promover o aprendizado de forma envolvente e acessível.

Desenvolvido exclusivamente com tecnologias web nativas, o site demonstra como criar uma aplicação completa utilizando apenas HTML, CSS e JavaScript puro.

---

## 🎯 Objetivos

- Incentivar a conscientização ambiental.
- Divulgar práticas agrícolas sustentáveis.
- Promover aprendizado por meio da gamificação.
- Demonstrar o uso de tecnologias web sem frameworks.

---

## 🏗️ Funcionalidades

### 🌿 Página Inicial

- Cabeçalho com o nome do projeto.
- Título principal e subtítulo.
- Introdução sobre sustentabilidade na agricultura.

### 🌎 Boas Práticas Sustentáveis

Cards informativos abordando:

- 💧 Economia de água
- ☀️ Uso de energia limpa
- 🌱 Preservação do solo
- ♻️ Reciclagem de resíduos
- 🦋 Proteção da biodiversidade

### 🧠 Quiz Interativo

O quiz possui:

- 10 perguntas de múltipla escolha.
- 4 alternativas por questão.
- Apenas uma resposta correta.
- Exibição de uma pergunta por vez.
- Barra de progresso.
- Contador de pontuação.
- Resultado final personalizado.
- Reinício sem recarregar a página.

---

## 📊 Sistema de Avaliação

Ao finalizar o quiz, o usuário recebe:

- Número de acertos.
- Percentual de desempenho.
- Mensagem personalizada.

### Possíveis Resultados

| Desempenho | Mensagem |
|------------|-----------|
| 90% - 100% | 🌟 Excelente |
| 70% - 89% | 🎉 Muito Bom |
| 50% - 69% | 👍 Bom |
| Abaixo de 50% | 📚 Continue Aprendendo |

## 🔧 Como Executar

### 1. Clone o repositório


### 2. Acesse a pasta do projeto
